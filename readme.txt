Moneroo enables businesses to instantly access multiple payment providers, primarily in Africa and globally, with a single WooCommerce integration.

== Description ==

Moneroo is a payment orchestration platform that allows businesses to offer a wide range of payment methods through a single integration.
This WooCommerce plugin brings the power of Moneroo to your WordPress eCommerce site.
You can give your customers multiple options to make payments without the hassle of manually integrating with each provider.
Keep your customers happy and increase your sales by offering them the payment methods they prefer.

Moneroo support :
* + 40 payment methods (mobile money, bank transfers, cards, crypto and more)
* + 30 countries
* + 10 currencies
* + 10 Payment providers

= Features =

* Single integration point for multiple payment providers.
* Supports a wide range of payment options, including mobile money, bank transfers, cards, and more.
* Supports multiple currencies.
* Supports multiple languages.
* Easy to set up within your WooCommerce store.
* Secure and reliable transaction methods.

== Installation ==

= From your WordPress dashboard =

1. Go to Plugins > Add New
2. Search for "Moneroo"
3. Click "Install Now"
4. Activate Moneroo from your Plugins page

= From WordPress.org =

1. Download Moneroo for WooCommerce Plugin.
2. Upload the 'moneroo-for-woocommerce' directory to your '/wp-content/plugins/' directory.
3. Activate Moneroo from your Plugins page.

= Configuration =

1. Navigate to WooCommerce > Settings > Payments
2. You'll see Moneroo listed among the available payment methods. Click 'Manage' next to it.
3. Configure your settings and enter your Moneroo API credentials.
4. Save changes.

== Frequently Asked Questions ==

= Does this support payment methods other than Africa local payment methods ? =

Yes, Moneroo supports a wide range of payment methods, focused primarily on the African market and expanding globally.

= Is it secure? =

Yes, the Moneroo platform uses industry-standard security protocols to ensure all transactions are securely processed.

= Does this require an SSL certificate? =

Yes! In Live Mode, an SSL certificate must be installed on your site to use Moneroo.

= Does this support both production mode and sandbox mode for testing? =

Yes, it does - production and Test (sandbox) mode is driven by the API keys you use with a checkbox in the Moneroo dashboard to toggle between both.

= Where can I find documentation? =
For help setting up and configuring, please refer to our [documentation](https://docs.moneroo.io/).

= Where can I get support or talk to other users? =

If you get stuck, you can ask for help in the Plugin Forum or via the [Moneroo Support](https://help.moneroo.io).
